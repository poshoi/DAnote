# 切片操作

## 1. 一维数组切片

&#x20;数组对象切片的参数设置与列表切面参数类似

**`数组对象[起始位置:终止位置:步长]`**&#x20;

`步长+`：默认切从首到尾\
`步长-` ：默认切从尾到头

<mark style="color:yellow;">切的范围为：\[起始位置：终止位置)</mark> &#x20;

```python
import numpy as np
a = np.arange(1,10)
print(a) # 123456789
print(a[:3]) # 123
print(a[3:6]) # 456
print(a[6:]) # 789
print(a[::-1]) #987654321
print(a[:-4:-1]) #987
print(a[-4:-7:-1]) #654
print(a[-7::-1]) # 321
```
